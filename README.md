# ID360 — Documentation d'intégration

Dépôt public hébergeant les guides d'intégration du SDK ID360 pour les applications mobiles Android et iOS.

Site : [id360docaposte.github.io](https://id360docaposte.github.io)

## Guides

| Plateforme | Document |
|---|---|
| Android | [GUIDE_INTEGRATION_ANDROID.md](GUIDE_INTEGRATION_ANDROID.md) |
| iOS | [GUIDE_INTEGRATION_IOS.md](GUIDE_INTEGRATION_IOS.md) |

Ces guides couvrent les prérequis, les modes d'intégration et les parcours recommandés selon votre cas d'usage (scan MRZ, lecture NFC, WebView, UI personnalisée).

## Utilisation

Les guides sont publiés dans les [releases GitHub](https://github.com/id360docaposte/id360docaposte.github.io/releases) et disponibles directement dans ce dépôt.
